# RESTProject

## Description
A RESTful web service project for managing DVD rental database (Sakila database).

## Technologies Used
- Java 19
- Jakarta EE 10
- JAX-RS (Jersey 3.1.9)
- JPA/Hibernate 6.2
- MySQL 8.0
- Maven
- Apache Tomcat 10.1.52

## Features
- CRUD operations for:
  - Actors
  - Films
  - Categories
  - Languages
  - Cities
  - Countries
  - Stores
  - Staff
  - Customers
